Important FAQ

What is Document Object Model (DOM)

DOM is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic. It defines the logical structure of documents and the way a document is accessed and manipulated

How do you select an element from the DOM?

To select an element from the DOM (Document Object Model) in JavaScript, there are several methods :
1.getElementById : Select an element using Id
2.getElementByClassName : Selects elements by their class name. This returns a live HTMLCollection.
3.getElementByTagName : Select elements by their tag name. This also returns a live HTMLCollection.
4.querySelectorAll: Selects all elements that match a CSS selector, returning a static NodeList.
A few more options are available.However this 4 are the most common methods !

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique used in the DOM to handle events on a group of elements by attaching a single event listener to a parent element, rather than attaching individual event listeners to each element. This approach takes advantage of the fact that events bubble up the DOM tree, allowing the parent element to capture and handle events triggered by its child elements.

How do you manipulate an element's attributes and styles using the DOM?

The DOM provides several methods for manipulating the properties and styles of an element. Some common ways to do this are getAttribute, setAttribute, style, removeAttribute etc. We can manipulate element's attributes through these methods.
